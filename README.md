# HelpDeck

HelpDeck is a multi-tenant, self-hostable AI customer support agent. Connect your
knowledge base (PDFs, URLs, raw text) and it answers customer questions **only**
from your own ingested docs — grounded RAG with inline source citations,
deterministic guardrails against hallucination, and automatic escalation to a
human when confidence is low. Ships as a FastAPI backend, a Next.js dashboard,
and an embeddable JS widget you can drop into any site with one script tag.

**Status: pre-alpha scaffold** — nothing to see here yet. See
[`docs/IMPLEMENTATION_PLAN.md`](docs/IMPLEMENTATION_PLAN.md) for the roadmap.
